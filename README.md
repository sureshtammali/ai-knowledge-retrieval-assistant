# AI-Powered Knowledge Retrieval Assistant for Intelligent Document Search

**Developed by:** Tammali Suresh

---

## Problem Statement

Team members often struggle to quickly find relevant information from internal documents, notes, and past project materials. As a result, knowledge is not reused effectively, work gets repeated, and onboarding new team members takes longer than it should.

---

## Problem Understanding

Organizations accumulate large volumes of documents over time — reports, meeting notes, project files, and manuals. Without an intelligent search system, employees waste hours manually scanning through files, leading to:

- Duplicated effort across teams
- Slower onboarding for new members
- Poor knowledge reuse and retention
- Inability to extract insights from existing materials quickly

---

## Proposed Solution

An AI-powered **Retrieval-Augmented Generation (RAG)** system that allows users to:

1. Upload any PDF document
2. Ask natural language questions about its content
3. Receive accurate, context-aware answers generated by a large language model

The system retrieves the most relevant sections from the document and uses **Cohere's command-r-plus** model to generate precise answers grounded in the document.

---

## How It Works

1. **Upload** a PDF document via the web interface
2. The document is **split into chunks** and indexed using TF-IDF vectorization
3. When a question is asked, the most **relevant chunks are retrieved** using cosine similarity
4. The retrieved context is passed to **Cohere's LLM** to generate a precise answer
5. **Chat history** is maintained within the session for multi-turn conversations

---

## Tech Stack

| Component | Technology |
|---|---|
| Frontend | Streamlit |
| PDF Parsing | PyMuPDF (fitz) |
| Text Vectorization | TF-IDF (scikit-learn) |
| Similarity Search | Cosine Similarity (numpy) |
| Language Model | Cohere command-r-plus-08-2024 |

## Setup & Installation

1. Clone the repository
2. Create and activate a virtual environment:
   ```
   python -m venv env
   env\Scripts\activate
   ```
3. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
4. Run the app:
   ```
   streamlit run src/app.py
   ```
5. Enter your **Cohere API key** in the sidebar (get one free at https://cohere.com)

## Evaluation

- Answers are grounded in actual document content, reducing hallucinations
- Fast retrieval using local TF-IDF — no external vector database required
- PDF is processed only once per session for efficiency
- Chat history maintained for multi-turn conversations
